## Setup
### ① Install modules
```
$ npm install
```

<br>

### ② Run ganache-cli
（Please make sure whether port number is `8545` or not）
```
$ ganache-cli
```

<br>

### ③ Compile contracts
```
$ npm run compile:local
```

<br>

### ④ Test contracts（※ In progress to implement）
```
$ npm run test:local
```
